=====================================================================
 GESTAO MAC MOBILE - UniFi (v4)  -  Pacote de distribuicao
=====================================================================

COMO FUNCIONA (v4):
 - Abre SILENT (sem janela preta) e abre o navegador sozinho.
 - Fecha sozinho quando o navegador e fechado.
 - Cada usuario faz login com a PROPRIA conta do UniFi.
 - A chave de criptografia (secret.key) e as credenciais (creds.enc) sao
   GERADAS LOCALMENTE em cada PC no primeiro login. NAO se distribui chave.
 - Banco compartilhado na rede (multiusuario). Logs na pasta logs\.

CONTEUDO DESTE PACOTE
  GestaoMAC.exe            -> o programa (silent)
  .env.coletor.exemplo     -> modelo de config (PC que tambem coleta)
  .env.usuario.exemplo     -> modelo de config (PC so visualizacao)
  banco_inicial\           -> history.db para "semear" a pasta de rede
  docs\                    -> manuais (MANUAL_TECNICO .md/.pdf, etc.)
  LEIA-ME.txt              -> este arquivo

---------------------------------------------------------------------
INSTALACAO MULTIUSUARIO (banco na rede) - passo a passo
---------------------------------------------------------------------
1) Crie uma pasta compartilhada na rede, ex.:
      \\SERVIDOR\Compartilhada\GestaoMAC\data\
   De permissao de LEITURA e ESCRITA aos usuarios.

2) Copie para essa pasta o arquivo de "banco_inicial\":
      history.db
   (Semeia o banco com os dados atuais. Faca so uma vez.)

3) EM CADA PC:
   - Copie GestaoMAC.exe para uma pasta local.
   - Renomeie um dos modelos para ".env" ao lado do exe e ajuste o DB_PATH
     para o caminho da pasta de rede:
        DB_PATH=\\SERVIDOR\Compartilhada\GestaoMAC\data\history.db
     - PC normal/coletor: .env.coletor.exemplo (COLLECT_ON_OPEN=1)
     - PC so visualizacao: .env.usuario.exemplo (COLLECT_ON_OPEN=0)
     - (Opcional) BRAND=NomeDaEmpresa para a marca no rodape.
   - NAO precisa copiar secret.key: o exe gera a dele no 1o login.

4) Abrir: duplo clique em GestaoMAC.exe -> abre o navegador.
   Login = a conta UniFi DO PROPRIO usuario daquele PC.
   Fechou o navegador -> o programa encerra sozinho.

---------------------------------------------------------------------
USO EM 1 SO PC (sem rede)
---------------------------------------------------------------------
- Copie GestaoMAC.exe para uma pasta e abra. No 1o login ele cria a base
  local (pasta data\) e a secret.key/creds.enc daquele PC.

OBS:
- Cada PC tem a propria secret.key e creds.enc (credenciais do usuario).
  Guarde/backup do history.db (na rede). NAO compartilhe secret.key/creds.enc.
- Backup: dentro do app, menu "Backup".
- Atualizacoes: substitua so o GestaoMAC.exe.
- Diagnostico: pasta logs\ (ao lado do exe).
